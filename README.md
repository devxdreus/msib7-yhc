# Tes Web Developer Digitaliz

## Tech Stack :
- Laravel 11
- Livewire & Volt
- Tailwind Css

## Fitur utama dalam aplikasi :
- Login & Register
- CRUD Kursus (Course)
- CRUD Materi (Lesson)
- Light & Dark Theme

## Cara Install

```bash
# clone this repository
git clone https://github.com/devxdreus/msib7-yhc

cd msib7-yhc

# install dependecies
composer install
npm install

# create .env file & configure the environment
cp .env.example .env

# migrate & seed data
php artisan migrate --seed

# run application
php artisan serve

# open new terminal
npm run dev
```
